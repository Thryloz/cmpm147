Description

Generate an adjustable look-alike to an inspiration image which changes to optimize an evaluation score with mutations and randomization.

Technical

1. The user is able to choose an image and shape from the drop down menu.
2. The design object is initalized depending on the shape, usually with randomized x and y coordinates, a number between 0 and 255 for fill, and in cases of the circle, a diameter.
3. The design is rendered, then the mutation function randomizes the shapes sizes and colors. This feeds back into an evaluation function giving the current design a score and comparing it to the best design, then adjusts the active image to be similar to the best image.
4. The mutation slider controls the size of the changes/mutations. At 1%, the changes are very minimal.
5. The intention is for the user to slowly move the slider back from 100% to 1% over the course of 30s, a process called annealing, to slowly fine-tune generator.
