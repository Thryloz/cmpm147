Experiment 02: Living Impressions

Description

Procedurally generate a snowy forest.

Technical

The background and snow were created with polygons similar to the example provided. The tree trunks are multiple lines connected together with the thickness slowly decreasing as it goes up the screen. The branches on the side and background are created recursively with some randomness to their angles. The snow is creating an array of points and their size in setup(), then drawing the ellipses in draw(). They're then moved down the screen and randomly left and right until they reach one of the edges and are moved back to the top/side. This project offers a generative, living impression inspired from these images:
[Source]
Relation to inspring image:
Color: The snow and tree colors were sampled from the inspiring image using the Krita eyedropper tool.
Primitives: The trees are rendered with line primitives.
Polygons: The two snow layers a single randomly polygon using vertex calls.
Composition: Different random seed values lead to differently positioned trees, differently shaped branches and snow.
Life: There's snow slowly falling and the whole image moves slightly with the mouse cursor.
